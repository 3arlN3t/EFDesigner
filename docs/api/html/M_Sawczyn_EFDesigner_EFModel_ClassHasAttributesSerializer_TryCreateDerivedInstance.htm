ClassHasAttributesSerializerTryCreateDerivedInstance Method

[This is preliminary documentation and is subject to change.]

This method creates a correct derived instance of ClassHasAttributes based on the tag currently pointed by the reader. Note that the difference between this method and the above one is that this method will never create an instance of the ClassHasAttributes type itself, only derived types are checked.
